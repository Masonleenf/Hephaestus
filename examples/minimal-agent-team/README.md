# Minimal Agent Team Example

This example shows the smallest public tree that still keeps Agentlas-style
memory, policy, and verification visible.

```text
example-team/
├── AGENTS.md
├── agents/
│   ├── 00-orchestrator/agent.md
│   └── 10-reviewer/agent.md
├── skills/
│   └── verification/SKILL.md
└── .agentlas/
    ├── agent-card.json
    ├── company-blueprint.json
    ├── memory-map.json
    └── memory-tickets.jsonl
```
